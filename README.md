# Python-Rock-Paper-Scissors
Game Rules
Match:   A Best of 3 competition between the player and a bot.
Winner:   The first to reach 3 points wins the game.
Options:   Choose between ROCK, PAPER, or SCISSORS.
Input:   Invalid choices don't count and let you try again.
Restart:   Option to start a fresh match or exit after the game ends.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Regras do Jogo
Partida: Uma disputa de Melhor de 3 entre o jogador e um bot.
Vencedor: O primeiro a atingir 3 pontos ganha o jogo.
Opções: Escolha entre PEDRA, PAPEL ou TESOURA.
Entrada: Escolhas inválidas não contam ponto e permitem tentar de novo.
Reinício: Opção de começar uma nova partida ou sair após o término.
